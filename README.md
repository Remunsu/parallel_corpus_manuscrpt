# Требования 
Python 3.8

# Установка
Cкачать и распаковать архив\
Открыть терминал в папке и выполнить следующие команды:\
python -m venv venv\
source venv/Scripts/activate\
pip install -r requirements.txt

# Запуск
Запустить app.py

# Описание функций
Списки можно загрузить по кнопке "Загрузить XML".\
По кнопке "Выбрать листы главного списка" можно выбрать определенные листы.\
В меню с названиями евангелий можно выбрать основное, нажав на него.\
Выравнивание строится по кнопке "Построить выравнивание".\
Изменить тип разночтения можно дважды нажав по типу в таблице.\
Сохранить таблицу можно по кнопке "Сохранить проект".\
Загрузить проект можно по кнопке "Открыть проект".\
Для фильтрации разночтений доступны чекбоксы в верхней части экрана.\
Справа присутствуют свойства слова, его контекст и положение в тексте.
